Seçilen CSS stilli şehre id'le sınıf ekle

Sayfa ilk yüklenirken "prompt" ileti kutusuyla şehir seçilmekte, sonra jsGw3.js dosyasının içerdiği hazır "w3.addClass(id,'sınıf')" fonksiyonuyla seçilen şehre sınıf eklenmektedir. Tüm şehirler "w3-red" CSS ön stille zemin-renklendirilmiştir.
