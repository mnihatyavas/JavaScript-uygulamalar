Dahiledilen/Gizlenen w3-CSS'li HTML

JS "w3.includeHTML()" fonksiyonu HTML p-paragraf içindeki "w3-include-html="jsG1001x.html" atıflı dosyayı birkez yükleyip belirtilen w3-CSS'li HTML sayfasını o konuma dahil etmektedir. Sonrasındaysa "id.style.visibility='visible'/'hidden'"le görünürlüğü göstermekte/gizlemektedir.
