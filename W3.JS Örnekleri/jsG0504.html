"TagName"le seçilen elemanın CSS sınıfını silme

Sayfa ilk yüklenirken "prompt" ileti kutusuyla HTML elemanı seçilmekte, sonra jsGw3.js dosyasının içerdiği hazır "w3.removeClass(tagName,'sınıf')" fonksiyonuyla CSS sınıfı silinmektedir.
