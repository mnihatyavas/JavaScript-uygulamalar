".class"la seçilen sınıfın CSS sınıfını silme/stilleme

Sayfa ilk yüklenirken "prompt" ileti kutusuyla ".class" seçilmekte, sonra jsGw3.js dosyasının içerdiği hazır "w3.toggleClass(".class",'sınıf')" fonksiyonuyla mevcut CSS sınıfı silmekte/stillemektedir. Br if(++sayaç%2) ise alternatif silinmeyi veya stillemeyi açıklamaktadır.
