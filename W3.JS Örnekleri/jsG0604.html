"tagName"le seçilen HTML eleman-larının sınıf-larını silme/stilleme

Sayfa ilk yüklenirken "prompt" ileti kutusuyla HTML eleman-ları seçilmekte, sonra jsGw3.js dosyasının içerdiği hazır "w3.toggleClass(tagName,'sınıf')" fonksiyonuyla CSS sınıfı silmekte/stillemektedir. Br if(++sayaç%2) ise alternatif silinmeyi veya stillemeyi açıklamaktadır.
