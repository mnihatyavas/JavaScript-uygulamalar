Dahiledilen/Gizlenen çoklu w3-CSS'li HTML

JS "w3.includeHTML()" fonksiyonu HTML div-bölüm içindeki "w3-include-html="jsG1001x.html" ve "jsG1003x.html" atıflı dosyaları birkez yükleyip belirtilen w3-CSS'li HTML sayfalarını o konuma dahil etmektedir. Sonrasındaysa "id1/2.style.display='block'/'none'"le görünürlüklerini göstermekte/gizlemektedir.
