".class"la seçilen sınıfın CSS sınıfını silme

Sayfa ilk yüklenirken "prompt" ileti kutusuyla sınıf seçilmekte, sonra jsGw3.js dosyasının içerdiği hazır "w3.removeClass('.class','sınıf')" fonksiyonuyla mevcut CSS sınıfı silinmektedir.
