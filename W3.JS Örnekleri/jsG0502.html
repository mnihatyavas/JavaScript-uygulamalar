"id"le seçilen şehrin CSS sınıfını silme

Sayfa ilk yüklenirken "prompt" ileti kutusuyla şehir seçilmekte, sonra jsGw3.js dosyasının içerdiği hazır "w3.removeClass(id,'sınıf')" fonksiyonuyla mevcut stilli CSS sınıf-ları silinmektedir.
