Seçilen sınıfa CSS sınıf ekle

Sayfa ilk yüklenirken "prompt" ileti kutusuyla sınıf seçilmekte, sonra jsGw3.js dosyasının içerdiği hazır "w3.addClass(.class,'sınıf')" fonksiyonuyla seçilen sınıfa CSS stilli sınıf eklenmektedir.
